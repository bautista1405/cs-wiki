# Appsmith

Appsmith is an open-source low-code platform that allows businesses to build custom software and internal tools significantly faster.

Key Features:
- Rapid Application Development: Build custom software 10X faster by reducing the need to code everything from scratch.
- Centralized IDE: Manage variables, functions, and logic in a built-in IDE with auto-complete and debugging.
- AI Copilots: Generate code and custom widgets using natural language prompts.
- Git Integration: Sync to git repositories for collaboration, version management, and CI/CD workflows.
- Open Source: Community Edition is available under the Apache 2.0 license.
- Enterprise Grade: Offers SAML/OIDC SSO, Role-based access controls (RBAC), audit logging, and SOC 2 Type II compliance.
- Deployment: Supports self-hosting with air-gapped deployment options.
