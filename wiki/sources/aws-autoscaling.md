# AWS Application Auto Scaling

AWS Auto Scaling monitors applications and automatically adjusts capacity to maintain steady performance at the lowest possible cost.

## Key Mechanisms
- **Dynamic Scaling**: Adjusts capacity in response to changing demand (e.g., based on CPU utilization).
- **Scheduled Scaling**: Scales capacity based on a known time-based pattern.
- **Predictive Scaling**: Uses machine learning to predict and stay ahead of demand.
