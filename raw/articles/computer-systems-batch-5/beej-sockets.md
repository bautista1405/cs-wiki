# Beej's Guide to Network Programming

A comprehensive guide to network programming using Internet sockets.

## Overview
- Covers the sockets API, which has been ported to Unix, Linux, and Windows.
- Provides practical implementation details for creating network connections.

## Available Formats
- HTML (single page, widescreen, or split)
- PDF (various versions including syntax highlighting and B&W)
