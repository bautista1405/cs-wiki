# NandGame - Build a computer from scratch

NandGame is an interactive educational tool that teaches computer architecture by having the user build components incrementally.

## Key Learning Path
The game starts with the simplest primitive (Nand gate) and guides the user through the construction of:
- Basic logic components.
- More complex circuitry.
- A fully programmable computer.

## Relation to Concepts
Closely aligns with the philosophy of [[nand2tetris-2026-nand2tetris]] and [[computer-architecture]].

## Bias Checks
- **Counter-arguments**: As a simplified game, it may abstract away critical physical constraints like propagation delay, capacitance, and power consumption.
- **Data gaps**: Does not provide deep theoretical proofs, focusing instead on the "how" of construction.
