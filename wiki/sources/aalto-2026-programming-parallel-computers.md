# Programming Parallel Computers (Aalto University)

## Summary
Course materials from Aalto University (CS-E4580) focusing on the necessity and practice of exploiting parallelism in modern computing. It emphasizes that without explicit awareness of parallel architectures, programmers may waste over 99% of a CPU's theoretical performance.

## Key Concepts
- **CPU Parallelism**: Modern CPUs utilize multiple cores, execution units, wide vector operations, and pipelining.
- **GPU Parallelism**: GPUs are evolved general-purpose processors requiring explicit code (e.g., C++/CUDA) to be utilized.
- **Performance Engineering**: Understanding assembly code, compiler behavior, and hardware execution to predict and optimize performance.
