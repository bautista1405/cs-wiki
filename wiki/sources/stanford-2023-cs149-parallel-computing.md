# CS149: Parallel Computing (Stanford)

## Summary
Stanford course (Fall 2023) providing a deep understanding of the fundamental principles and engineering trade-offs in modern parallel computing systems.

## Key Topics
- **Multi-core processors**: SIMD, multi-threading, latency vs. bandwidth.
- **GPU Architecture**: CUDA programming abstractions and implementation.
- **Distributed Computing**: Data-parallel operations (map, reduce, scan) and Spark.
- **Memory Systems**: Cache coherence (MSI/MESI), memory consistency models, and DRAM.
- **Synchronization**: Fine-grained locks, lock-free programming (ABA problem, hazard pointers).
- **Hardware Specialization**: FPGAs, mobile SoCs, and energy-efficient computing.
