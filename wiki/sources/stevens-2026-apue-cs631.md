# Advanced Programming in the UNIX Environment (Stevens/Schaumann)

## Summary
Course (CS631) teaching complex system-level software development in C, using NetBSD as the primary reference platform. Based on the textbook by W. Richard Stevens.

## Topics Covered
- **User/Kernel Interface**: System calls and the UNIX programming environment.
- **I/O and Filesystems**: Advanced I/O, file sharing, and directory management.
- **Process Management**: Process relationships, signals, and interprocess communication (IPC).
- **Development Tools**: Use of `make`, `gdb`, `strace`, and `ltrace`.
