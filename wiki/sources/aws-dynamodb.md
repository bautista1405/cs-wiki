# Amazon DynamoDB

A key-value and document database that delivers single-digit millisecond performance at any scale.

## Scaling Features
- **On-Demand Mode**: Automatically scales throughput capacity up or down to meet application traffic.
- **Provisioned Capacity**: Allows developers to specify the number of reads and writes per second.
- **Global Tables**: Multi-region, multi-active database for low-latency access globally.
