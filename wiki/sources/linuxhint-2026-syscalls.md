# List of Linux System Calls

A reference guide to the primary interface between user-space applications and the Linux kernel.

## Key Takeaways
- Provides a mapping of system calls to their functional categories (file management, process control, network etc).
- Essential for understanding how software requests resources from the hardware via the kernel.

## Bias Checks
- **Counter-arguments**: A list of syscalls provides the "what" but not the "how" or "why" of kernel implementation.
- **Data gaps**: Lack of architectural context (e.g., how syscalls differ between x86 and ARM).
