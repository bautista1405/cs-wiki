# Summary: Operating Systems: Three Easy Pieces (OSTEP)

**Operating Systems: Three Easy Pieces (OSTEP)** by Remzi and Andrea Arpaci-Dusseau is a fundamental text on operating system design. It organizes the complexity of OSes into three central pillars:

1. **Virtualization**: How the OS abstracts hardware (CPU and memory) to create an illusion of private resources.
2. **Concurrency**: How the OS manages multiple threads of execution without corruption.
3. **Persistence**: How the OS ensures data survives crashes through stable storage (filesystems).

It covers essential topics such as CPU scheduling and memory management in detail.
