# CS143: Compilers (Stanford)

## Summary
Stanford course on the design and implementation of compilers, utilizing the "Cool" language project.

## Curriculum
- **Front-end**: Lexical analysis, parsing (top-down and bottom-up), and syntax-directed translation.
- **Semantic Analysis**: Type checking and run-time environments.
- **Back-end**: Code generation, intermediate code, local and global optimization, and register allocation.
- **Memory**: Automatic memory management.
