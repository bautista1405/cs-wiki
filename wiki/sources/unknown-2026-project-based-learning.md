# Project-Based Learning

## Summary
A curated list of tutorials and resources for learning programming through the creation of actual projects.

## Key Takeaways
- Learning is most effective when applied to real-world projects across various languages (JS, Python, Go, C++).
- Availability of numerous platforms (CodeCrafters, Exercism, etc.) that facilitate this approach.

## Bias Check
- **Counter-arguments**: Project-based learning can lead to "tutorial hell" where the learner copies code without understanding the underlying theory.
- **Data gaps**: The source is a list of links and does not provide a pedagogical framework for how to implement project-based learning effectively.
