# Obsidian - Sharpen your thinking

The free and flexible app for your private thoughts.

Obsidian is a powerful knowledge base that works on local Markdown files. It allows users to create a web of interconnected notes, building a personal wiki.

Key Features:
- Local storage: Notes are saved as plain text Markdown files on your device.
- Linking: Create links between notes to build a knowledge graph.
- Community Plugins: Extend functionality with a vast library of community-created plugins (e.g., Calendar, Kanban, Dataview).
- Sync: End-to-end encrypted synchronization across devices.
- Publish: Turn notes into an online wiki or digital garden via Obsidian Publish.
- Canvas: A spatial tool for visualizing ideas.
- Web Clipper: Easily capture content from the web.
